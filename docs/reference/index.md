# CLI Reference

The `hpx` CLI has four subcommands:

| Command | Description |
|---|---|
| [`hpx profile`](profile.md) | Profile a model on target hardware |
| [`hpx doctor`](doctor.md) | Check toolchain and dependencies |
| [`hpx boards`](boards.md) | List supported boards and SoC capabilities |
| [`hpx engines`](engines.md) | List available inference engines |

## Global Options

```
hpx --version    Show version and exit
hpx --help       Show help
```
