## Requirements

- Python >= 3.10
- ARM GCC toolchain (`arm-none-eabi-gcc`)
- CMake >= 3.24 and Ninja
- SEGGER J-Link tools
- [neuralspotx](https://github.com/AmbiqAI/neuralspotx) (`nsx` CLI)

## Install

```bash
pip install helia-profiler
```
